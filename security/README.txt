Poursuivez avec la page login, où vous pourrez tester les mécanismes de connexion.
Accédez ensuite à la page comments pour voir les fonctionnalités de commentaire et leurs vulnérabilités.
Terminez par la page upload pour explorer les problèmes liés au téléchargement de fichiers.
Liste des vulnérabilités par page :

Registre : Injection SQL
Se connecter : Injection SQL, attaques par force brute
commentaires : Vulnérabilité XSS
télécharger : Problèmes de sécurité liés aux fichiers téléchargés
